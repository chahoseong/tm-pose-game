
게임 이름: Fruit Catcher

기본 구조:
* 화면은 LEFT / CENTER / RIGHT 3개 구역으로 구성됩니다.
* 플레이어는 포즈 인식으로 바구니(캐릭터)를 이동시켜 하늘에서 떨어지는 과일을 받습니다.

조작 방식 (포즈):
* 왼쪽 기울이기 (또는 왼손 들기) → LEFT 이동
* 차려 자세 (또는 만세) → CENTER 이동
* 오른쪽 기울이기 (또는 오른손 들기) → RIGHT 이동

아이템 종류:
1. 사과 (Apple) → +100점
2. 황금 사과 (Gold Apple) → +300점 (드물게 등장)
3. 폭탄 (Bomb) → 닿으면 즉시 게임 오버

게임 규칙:
* 과일이 바닥에 닿기 전에 받아야 점수가 오릅니다.
* 폭탄을 받으면 게임이 종료됩니다.
* 시간이 지날수록 과일이 떨어지는 속도가 빨라집니다. (20초마다 레벨 업)

UI 요소:
* 상단: 현재 점수, 레벨 표시
* 중앙: 게임 화면 (캐릭터, 떨어지는 아이템)
* 하단: 포즈 인식 상태 모니터
